# Dawn Launcher - Complete Stability, UI, Functionality, and Minecraft Launching Overhaul

You are working on Dawn Launcher. This is not a prototype, demo, mockup, or vibe-coded project. Treat this as a production-grade Minecraft launcher.

## Core Requirements

### 1. Persistent Launch History System

Implement a persistent launch history system.

For every Minecraft launch:

* Store launch timestamp
* Store game version
* Store loader type (Vanilla, Fabric, Forge, NeoForge, Quilt)
* Store launch duration
* Store exit code
* Store account used
* Store instance launched

History must persist between launcher restarts.

The statistics cards must use real data from launch history:

* Last Played
* Total Play Time
* Number of Launches
* Most Played Instance
* Most Used Account
* Recent Activity

No fake values.

No placeholders.

No generated sample data.

Everything must reflect actual usage.
